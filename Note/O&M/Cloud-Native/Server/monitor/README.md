### Why monitor

- 保障业务的正常运行

## 监控的两个部分

### 1. 基础设施——USE

- U：Utilization	使用率
- S：Saturation        饱和度
- E：Error                  错误



##### e.g.

- CPU
  - 使用率
  - 负载
- mem
  - 使用率
- disk
  - 使用率
  - IO
- network
  - 使用率
  - 饱和度
  - 错误数

### 2. 业务系统——The Four Golden Signals

- latency：延迟
- traffic：流量
- errors：错误
- saturation：饱和度

## 监控的两个模式

- pull
  - 监控系统服务端与agent直连
- push
  - 监控系统服务端无法 与agent直连
